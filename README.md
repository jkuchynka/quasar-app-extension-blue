Blue - Quasar App Extension
===

Currently includes a confirm component, as well as a data table component

# Install
```bash
quasar ext add blue
```
Quasar CLI will retrieve it from NPM and install the extension.

# Uninstall
```bash
quasar ext remove blue
```
